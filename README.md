## 📋 Overview

This repository contains hands-on materials for evaluating African Automatic Speech Recognition (ASR) systems using linguistically grounded, robustness-aware methods. Designed for the Deep Learning Indaba 2026 tutorial "Sovereign Evaluation: Stress-Testing African ASR Systems Beyond WER."

### What You'll Learn
- ✅ Run inference with multiple ASR models on African language data
- ✅ Compute and interpret WER, CER, and diagnostic metrics
- ✅ Conduct structured error analysis (omissions, tonal errors, code-switching)
- ✅ Stratify evaluation by domain, year, and complexity
- ✅ Design sovereign evaluation pipelines for local deployment
